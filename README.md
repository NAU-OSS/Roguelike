Contributions are what make open source projects thrive. Whether you're fixing a typo, squashing a bug, or designing a whole new dungeon floor, your help is welcome here.

### How to Contribute

1. **Fork** the repository
2. **Create a branch** for your feature
3. **Make your changes** and write tests if applicable
4. **Run the test suite** to make sure nothing is broken
5. **Commit** your changes
6. **Push** to your fork
7. **Open a Pull Request** â€” describe what you changed and why

### Code of Conduct

This project is committed to providing a welcoming, inclusive environment for all contributors. Please be respectful and constructive in all interactions. Harassment or exclusionary behavior of any kind will not be tolerated.

### Reporting Bugs
